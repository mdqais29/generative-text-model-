## Overview
This project implements a generative text model using a pretrained GPT-based transformer from HuggingFace Transformers. The system accepts user prompts and generates coherent paragraphs based on the input topic.

The model performs autoregressive text generation, meaning it predicts the next word step-by-step to form meaningful sentences and structured paragraphs.

This notebook demonstrates how modern large language models can be used for natural language generation tasks.

## Objective
- Load a pretrained GPT model
- Accept user prompts
- Generate coherent paragraphs
- Demonstrate transformer-based text generation
- Showcase NLP inference using PyTorch

## Features
- Text generation using GPT-2
- Tokenization and decoding
- Temperature and sampling control
- GPU support (if available)
- Interactive notebook-based input
